# Setup

* Download txt file can be found [here](https://github.com/dwyl/english-words/)
* Rename the txt file into "words_alpha.txt"
* Uncomment the `get_5_letter_words` line in the `main` function in `main.cpp`
* Compile and run the program to get the `words5.txt`
* Comment out again the `get_5_letter_words`
* Compile and run the program to use
